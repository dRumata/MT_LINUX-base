Решение

  1. Получите информацию о встроенной в bash команде printf.
help printf
  1. Получите информацию о внешней команде printf. Есть ли разница между встроенной и внешней командами printf и какая?
man printf
  1. Узнайте версию внешней команды printf.
$(which printf) --version

Cделайте скриншот экрана c результатом выполнения задания для отправки отчета.

  1. Какой параметр обычно используется для получения краткой помощи по командам? Используя этот параметр, получите краткую помощь по внешней команде printf.
/usr/bin/printf --help
  1. Изучите краткую справку и man-страницу. Где можно получить дополнительную информацию о команде printf в системе?
info '(coreutils) printf invocation'
  1. Изучив способы указания формата выводимых данных, выведите командой printf значения переменных, в которых хранятся имя пользователя и его UID. При этом, для имени пользователя (строка символов) выделите 15 позиций, а для UID (целое число) - 10. Выровняйте вывод по левому краю.
printf '%-15s %-10d\n' $USER $UID
  1. Найдите все man-страницы с именем hostname.
man -f hostname
  1. Выведите список всех страниц, в названии и кратком описании которых присутствует hostname.
man -k hostname
  1. Выведите только список команд, в названии и кратком описании которых есть hostname.
man -k hostname -s 1,8
